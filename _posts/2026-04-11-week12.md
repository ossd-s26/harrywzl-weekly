## opensource in industry
In class we discussed how the tech industry shifted its approach to open source software. 
Companies originally adopted open source to save money and resources. Now, they use it as a strategy for collaboration.
Many businesses rely on open source tools for their operations, so they pay employees specifically to maintain these projects. Open source can also be profitable. Companies like Red Hat build their business models around supporting free software.

## individual contribution
I am looking into contributing to a version control system called jj. 
I prefer it over Git for several reasons. It treats the working tree as a commit, which removes the need to stash changes. 
It allows users to commit and push unresolved merge conflicts to avoid blocking progress. 
The system also includes an undo command to revert actions. It acts as a frontend for Git repositories, so I can use it locally without affecting my teammates.
